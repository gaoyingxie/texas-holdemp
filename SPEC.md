# 德州扑克 — SPEC.md

## 1. TDD 视角：这些行为先写测试

**扑克牌基础**
- `[ ]` 52张牌，花色×面值不重复
- `[ ]` 洗牌后随机发牌
- `[ ]` 发牌均匀：每人2张，桌面5张

**比大小**
- `[ ]` 同花顺 > 四条 > 葫芦 > 同花 > 顺子 > 三条 > 两对 > 一对 > 高牌
- `[ ]` 同 rank 时比踢脚（Kickers）
- `[ ]` 花色不比大小（平局时庄家胜）

**下注轮次**
- `[ ]` Pre-flop：每人发2张，押盲注
- `[ ]` Flop：翻3张公共牌，下注
- `[ ]` Turn：翻第4张，下注
- `[ ]` River：翻第5张，下注
- `[ ]` Showdown：摊牌比大小

**行动选项**
- `[ ]` Fold（弃牌）：退出本局
- `[ ]` Check（过牌）：不下注过
- `[ ]] Call（跟注]：下注等于当前最低
- `[ ]` Raise（加注]：高于当前最低
- `[ ]` All-in（全压]：压上全部筹码

**游戏流程**
- `[ ]` 小盲/大盲轮换
- `[ ]` 玩家初始筹码一致
- `[ ]` 某家筹码归零 → 游戏结束
- `[ ]` 可以开始新游戏

**AI 对手**
- `[ ]` 每轮随机行动（Fold/Check/Raise）
- `[ ]` 拿到好牌（对子以上）更倾向于跟/加

## 4. 游戏规则速查

| 阶段 | 事件 |
|---|---|
| Pre-flop | 发2张底牌，押大小盲 |
| Flop | 发3张公牌，第一轮下注 |
| Turn | 发第4张公牌，第二轮下注 |
| River | 发第5张公牌，第三轮下注 |
| Showdown | 剩余玩家摊牌，比5张最优 |

**手牌强度（从大到小）：**
1. 皇家同花顺（AKQJ10同花）
2. 同花顺
3. 四条
4. 葫芦（三带二）
5. 同花
6. 顺子
7. 三条
8. 两对
9. 一对
10. 高牌

## 5. UI 交互

- 点击卡牌区域选中/取消
- 按钮：Fold / Check / Call / Raise
- Raise 点击后弹出输入框（或滑动条）
- 回合结束自动过渡
